CSS Positioning

If you're looking at the image above and thinking 'Wait, why would you use an image of such poorly aligned things in an article about positioning?' then clearly you have yet to experience the bittersweet joys of CSS. If this is indeed your first venture into the world of CSS positioning then know this, it will NOT be easy. In-fact you're likely to give serious consideration to setting fire to your computer and running away to live a tech free life in the forest before all this is over. Rest assured though, CSS has been used to style practically every page on the web for a good fifteen years now and despite it's awkward and fiddly nature it really does work once you get the hang of it.

The first thing you're going to need to get comfortable with if you're going to make your layout work is the different types of positioning built into CSS. Any HTML object can be assigned a 'position' property within your CSS style sheet. These position properties come in four different types: static, relative, absolute and fixed.
